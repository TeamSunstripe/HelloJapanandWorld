NSObject

Class/クラス

ほとんどのObjective-Cクラス階層のルートクラス。ルートクラスからサブクラスがランタイムシステムへの基本的なインターフェイスとObjective-Cオブジェクトとして動作する機能を継承します。

 The root class of most Objective-C class hierarchies, from which subclasses inherit a basic interface to the runtime system and the ability to behave as Objective-C objects.

@interface NSObject

概要

AppKitには、macOSアプリのユーザーインターフェイスを実装するために必要なすべてのオブジェクト（ウィンドウ、パネル、ボタン、メニュー、スクローラー、テキストフィールド）が含まれており、画面に効率的に描画し、ハードウェアと通信するため、すべての詳細を処理します デバイスと画面バッファ、描画する前に画面の領域をクリアし、ビューをクリップします。 フレームワークは、障害を持つユーザーがアプリにアクセスできるようにするために使用するAPIも提供します（Accessibilityを参照）。 アプリをさまざまな言語、国、または文化的地域にローカライズする方法の詳細については、「国際化およびローカリゼーションガイド」を参照してください。
