JavaScriptとは？

JavaScriptは、米Netscape Communications社（当時）とSun Microsystems社が共同で開発したWebページを操作するためのプログラミング言語です。 活用することによって、これまでのHTML（Hyper Text Markup Language）だけでは不可能だったインタラクティブなWEBページが作成できるようになりました。

JavaScriptの書き方

JavaScriptファイルは基本的に.jsの拡張子で作成します。

test.js

// JavaScript HTML 側に表示する場合
document.write("Hello,JavaScript World!\n");
// コンソール に表示する場合
console.log("Hello,JavaScript World!\n");
// オリジナルコンソール に表示する場合
displayConsole("Hello,JavaScript World!\n");

サンストライプのお知らせ

IT／WEBエンジニアに特化した学習サイト「Hello World!!」。 プログラミングスキルチェックであなたのプログラミング力をみんなでチェックすることを目的にしていきます。
自分の学習のチェックをアウトプット可視化し、スキルも含めみんなでチェックしていけばいくほど、現場とほぼ同じスキルをマスターしていけるという無料コンテンツです。
これからIT／WEBエンジニアとして就職や転職をしたいと思っている方には、かなり有利になる画期的なサービスです。
さらに、プログラミング初心者向けの学習方法として、それぞれのプロフィール（紹介ページ）を作成して、自分自身のポートフォリオを作成していくことも目的にしていくので
みんなのスキルやみんなの成長がみんなで確認できるのである種成長ゲームのようにコンテンツを活用してほしいと思います。
